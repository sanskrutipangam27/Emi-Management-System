<h1>The EMI Management System</h1> 
<p>The EMI Management System is a web-based application designed to help users manage and track their EMI (Equated Monthly Installment) payments.
It allows users to store EMI details, monitor payment status, and organize financial records in a structured way.
This system helps simplify EMI tracking and ensures that users do not miss their installment payments.</p>

_**Features:**_
<ul>
<li> Add new EMI records</li>
<li>View all EMI details</li>
<li>Track monthly installment payments></li>
<li>Update EMI payment status></li>
<li>Manage borrower information></li>
<li>Organized record keeping</li>
</ul>

_**Technologies Used:**_
<ul>
<li>HTML – Structure of the web pages</li>
<li>CSS & Bootstrap5 – Styling and layout</li>
<li>JavaScript – Client-side functionality</li>
<li>Node.js – Server-side logic</li>
<li>NeDB – Database for storing EMI records</li>
</ul>

_**System Modules:**_

_Admin Module_:
<ul>
<li> Manage EMI records </li>
<li> Donwload the records </li>
<li> View all borrower details </li>
</ul>

_User Module_:
<ul>
<li> Add EMI information</li>
<li> Track installment payments</li>
<li> View payment history</li>
</ul>

**_How the System Works:_**
<ul>
<li>User enters EMI details such as amount, interest rate, and duration.</li>
<li>The system stores the data in the database..</li>
<li>Users can view all EMI records and track monthly payments..</li>
<li>Admin can see the payment status and records and can download them in csv file.</li>
</ul>

_**Advantages**_
<ul>
<li>Easy EMI tracking</li>
<li>Organized financial records</li>
<li>Reduces chances of missing payments</li>
<li>Simple and user-friendly interface</li>
</ul>
